# Viterbi Decoder

本项目将MATLAB中的`vitdec`函数使用MATLAB Coder功能生成C代码，并通过Python中的`ctypes`包装成Python包给Python应用程序使用。

## Compile

```sh
make
```

## Usage

查看 tests/test.py 获取使用viterbi decoder的例子。
